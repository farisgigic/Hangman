# Hangman Game - Scala

## Overview
Hangman is a timeless word-guessing game that challenges players to guess a hidden word one letter at a time. This project implements the game in Scala, combining functional programming principles with interactive gameplay. The game supports both single-player and multiplayer modes, includes a scoreboard, and allows saving and resuming progress.

## Features
- **Single-Player Mode**: Random word selection for players to guess.
- **Multiplayer Mode**: Players can input custom words for others to guess.
- **Scoreboard**: Tracks player scores and displays them in descending order.
- **Save and Resume**: Players can save their progress and continue the game later.
- **Interactive Interface**: Displays the current state of the word and dynamic hangman stages.

## Research
To develop this project, various implementations of Hangman in other programming languages were studied:
- **Python**: Console-based games focusing on simplicity and readability.
- **Java**: Object-Oriented Programming with GUI enhancements using Swing or JavaFX.
- **JavaScript**: Web-based implementations with HTML and CSS for interactivity.
- **C++**: Console-based games emphasizing memory management and control flow.
- **Scala**: Leveraging functional programming concepts like recursion and immutability.

## Learning Outcomes
This project provided a valuable learning experience in:
- Functional programming and recursion in Scala.
- File handling for saving and loading game states.
- Creating modular and maintainable code.
- Designing user-friendly interfaces and engaging gameplay mechanics.
